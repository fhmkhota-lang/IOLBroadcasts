Deploy to GitHub Pages

This portal is a static site — zero backend required. Deploy it in 3 steps:

  1. Create a new GitHub repository (e.g. iol-broadcasting)
  2. Upload all files from this project to the repo root
  3. Go to Settings → Pages → Source: main branch / root → Save

Your portal will be live at https://yourusername.github.io/iol-broadcasting
